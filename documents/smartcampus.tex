\section{Présentation du projet}
\subsection{Notre projet}
\begin{frame}{\insertsubsection}
\begin{center}\textbf{But du projet :} améliorer l'accessibilité et l'utilisabilité du campus, et centraliser les informations.\\\end{center}
\vspace{0.2cm}
Réalisation de différents outils :
\begin{itemize}
\item Application mobile de réalité augmentée (AR) ;
\item Application administrateur ;
\item Application super-administrateur ;
\item Maquette du campus de Grenoble.
\end{itemize}
\end{frame}
